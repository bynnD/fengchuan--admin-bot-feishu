# feishu--admin-bot
used for creating approval sheet
为了让您更方便地提交行政类申请，我们已经上线了飞书行政助理机器人。以下是从您的视角出发的完整使用流程说明，供参考。

一、开始对话 您只需要在飞书中向机器人发送一句话，例如：“我要采购一台笔记本”或“帮我用印一个合同”。机器人会自动识别您想办理的审批类型，支持一条消息包含多个申请。

二、补充必要信息 如果申请需要填写的字段不完整，机器人会提示您补充缺少的信息。您可以用自然语言直接补充，机器人会自动记住并合并信息，不会重复询问。

三、规则提醒 机器人会根据公司规则进行合规性判断，例如采购金额过高等情况，会提示相应的注意事项或风险说明。

四、当前支持的审批类型 用印申请、采购申请（均通过 API 自动提交）。

五、自动提交 当信息齐全且无需附件跳转时，机器人会直接替您提交审批，并返回一条确认消息，包含您提交的关键信息和行政审核意见。

六、后续流程 提交完成后，您只需等待主管审批即可，机器人会提示您“等待主管审批”。

如需新增审批类型或调整规则，管理员可通过配置文件快速修改，无需改动核心流程。

如有更多问题，欢迎随时继续提问。

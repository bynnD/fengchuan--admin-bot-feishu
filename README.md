# feishu--admin-bot
used for creating approval sheet
为了让您更方便地提交行政类申请，我们已经上线了飞书行政助理机器人。以下是从您的视角出发的完整使用流程说明，供参考。

一、开始对话 您只需要在飞书中向机器人发送一句话，例如：“我想请假”或“我要提交采购申请”。机器人会自动识别您想办理的审批类型。

二、补充必要信息 如果申请需要填写的字段不完整，机器人会提示您补充缺少的信息。例如请假需要开始日期、结束日期、天数和原因等。您可以用自然语言直接补充，机器人会自动记住并合并信息，不会重复询问。

三、规则提醒 机器人会根据公司规则进行合规性判断，例如请假天数过长、采购金额过高等情况，会提示相应的注意事项或风险说明。

四、附件必填场景 若某个审批在飞书原生表单中设置了“附件必填”，机器人会自动识别并直接给您发送跳转链接，引导您进入飞书原生审批页面上传附件并完成提交。

五、自动提交 当信息齐全且无需附件跳转时，机器人会直接替您提交审批，并返回一条确认消息，包含您提交的关键信息和行政审核意见。

六、后续流程 提交完成后，您只需等待主管审批即可，机器人会提示您“等待主管审批”。

如需新增审批类型或调整规则，管理员可通过配置文件快速修改，无需改动核心流程。

如有更多问题，欢迎随时继续提问。
